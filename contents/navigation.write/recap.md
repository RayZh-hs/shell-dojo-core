## RECAP

在本章中, 你学习/练习了以下命令:

- `cp` / `cp -r`: 复制文件/目录;
- `mv`: 移动或重命名文件/目录;
- `rm` / `rm -r` / `rm -rf`: 删除文件/目录, 注意风险.
- `mkdir` / `mkdir -p`: 创建单个/多级目录;

### 指令速查表

- `cp a.txt b.txt` : 复制文件;
- `cp -r src dst` : 复制目录;
- `mv a.txt b.txt` : 重命名;
- `mv a.txt dir/` : 移动至目录;
- `rm file` / `rm -rf dir/` : 删除文件/强制删除目录.
- `mkdir project && cd project` : 新建后进入;
- `mkdir -p a/b/c` : 递归创建多级目录;
