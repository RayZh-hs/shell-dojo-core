---

`cd` 全称 `Change Directory`,顾名思义,就是切换目录.

`cd dir` 会进入目录 `dir` ,如果不写目录,则回到主目录.这里列举了几种 `cd` 的常见用法:

```
cd ~            # 进入主目录
cd ~/Desktop    # 跳到桌面目录
cd ..           # 返回上一级目录
cd -            # 返回上一次访问的目录,像“返回”按钮一样.
```
